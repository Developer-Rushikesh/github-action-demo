# 🚀 GitHub Actions CI/CD Pipeline Project

## 📌 Project Overview

This project demonstrates a simple CI/CD pipeline using GitHub Actions. A static website built with HTML and CSS is automatically deployed to GitHub Pages whenever code is pushed to the repository.

## 🎯 Objective

The objective of this project is to understand:

* Git and GitHub workflow
* YAML configuration
* GitHub Actions
* Continuous Integration (CI)
* Continuous Deployment (CD)
* GitHub Pages Hosting

---

## 🛠 Technologies Used

* HTML5
* CSS3
* Git
* GitHub
* GitHub Actions
* YAML
* GitHub Pages

---

## ⚙ Workflow

```text
Developer Push Code
        ↓
GitHub Actions Triggered
        ↓
Checkout Repository
        ↓
Build & Validate
        ↓
Deploy to GitHub Pages
        ↓
Live Website Updated
```

## 🚀 GitHub Actions Workflow

The workflow automatically runs when code is pushed to the main branch.

Main steps:

1. Checkout repository code
2. Configure GitHub Pages
3. Upload website files
4. Deploy website automatically
